# DeepLearningFromScratchPy2
python2 conversion of deep-learning-from-scratch for compuational linguistics class at SNU by Hyopil Shin

Original Github for the textbook is in https://github.com/WegraLee/deep-learning-from-scratch

소스 코드를 실행하려면 아래의 소프트웨어가 설치되어 있어야 합니다.

    파이썬 2.x
    NumPy
    Matplotlib

# 원래 deep-learning-from-scratch에서 제공하는 코드들은 챕터별로 되어 있지만 여기서는 common과 dataset을 제외한 각 챕터별 코드들을 mine에 모아 놓았습니다. 해당 코드앞에 챕터 표시를 해놓았습니다
# mnist data를 가져오는 dataset/mnist.py를 수정하였기 때문에 코드들을 실행하기 위해서는 맨처음 /dataset/mnistPy2.py 파일을 실행하여, mnist 데이터를 numpy로 변환하고 python2에 맞는 pickle을 생성하여야 합니다.
